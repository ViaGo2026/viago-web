# viago-web
AWS 기반 글로벌 여행 후기 웹 서비스

<img width="1007" height="879" alt="image" src="https://github.com/user-attachments/assets/04be2de1-8d2c-49c4-97b8-b92e185d1969" />


1. 글로벌 사용자를 고려한 멀티 리전 웹서비스 구축
2. ALB와 Auto Scaling Group을 활용한 고가용성 확보
3. RDS MariaDB를 활용한 데이터 계층 분리
4. S3와 CloudFront를 활용한 이미지·영상 제공 최적화
5. OpenVPN을 통한 안전한 관리자 접속 환경 구축
6. ACM, WAF, Security Group을 활용한 보안 강화
7. CloudFormation을 활용한 인프라 자동화
8. Comprehend, Rekognition을 활용한 광고, 음란물 자동 검열
